# Configuración de QR por dispositivo

Sube todo el contenido de esta carpeta al repositorio de GitHub Pages. Cada dispositivo debe abrir el juego con su parámetro `device`.

Ejemplo: `https://TU_USUARIO.github.io/TU_REPO/?device=roy-riviera`
